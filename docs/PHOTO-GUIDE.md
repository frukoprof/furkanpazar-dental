# Fotoğraf Çekim Rehberi

Site için klinik ve hekim fotoğrafları. Telefonla çekilebilir; önemli olan kurallara uymak.

## ⚠ Değişmez kurallar (mevzuat + KVKK)

- **Hiçbir karede hasta olmasın** — yüz, ağız, el, hasta sırası dahil. Öncesi-sonrası de YOK.
- **Ekranda/masada hasta adı, dosya, röntgen görünmesin** (kişisel veri).
- **Filtre/rötuş yok** — olduğu gibi. (Estetik düzenleme yönetmelikte sorun yaratır.)
- Her sahneden **3-4 farklı açı** çek; en iyisini ben seçerim.

## 📱 Genel ayarlar (her çekimde)

- Telefonu **en yüksek çözünürlükte** kullan, **lensi temiz bez**le sil.
- **Izgara (grid)** aç → yatay/dikey çizgileri düz tut (eğri fotoğraf amatör durur).
- **Gündüz + doğal ışık**: perdeleri aç, tavan lambalarını da yak. **Flaş KULLANMA.**
- Ortamı **topla**: kablo, kağıt, bardak, kişisel eşya, çöp kovası görünmesin.
- **Yatay (manzara) çek** — galeri için idealdir. Sadece portrede dikey de gerekiyor.
- Gönderirken **sıkıştırma**: WhatsApp'ta "Belge olarak gönder" ya da e-posta ile yolla (normal WhatsApp fotoğrafı kaliteyi düşürür).

## 1. Hekim Portresi  ⭐ (en önemli kare)

| | |
|---|---|
| **Yön** | Hem **dikey** (mobil için) hem 2-3 **yatay** çek |
| **Çerçeve** | Göğüs hizasından yukarı; başın üstünde biraz boşluk kalsın |
| **Işık** | Yüzün **pencereye dönük** olsun (ışık yüze gelsin). Arkanda pencere OLMASIN (yüz kararır) |
| **Arka plan** | Sade duvar veya hafif bulanık klinik; dağınık raf/pano olmasın |
| **Kıyafet** | Önlük; düz renk tercih |
| **Poz** | Doğal, hafif gülümseme; kolları kavuşturma. Biri ciddi-nötr, biri gülümseyen 2 versiyon |
| **İpucu** | Birinin telefonu tutup 4-5 kare çekmesi, aynadan/sehpadan daha iyi sonuç verir |

## 2. Tedavi Odası

| | |
|---|---|
| **Yön** | Yatay |
| **Kompozisyon** | Odanın **köşesinden** geniş açı; ünit ve koltuk toplu görünsün |
| **Düzen** | Koltuk düz konumda, aletler örtülü/düzenli, tezgah temiz |
| **Işık** | Tüm lambalar açık; gölge az olsun |

## 3. Sterilizasyon / Hijyen  ⭐ (güven için çok değerli)

| | |
|---|---|
| **Ne** | Autoklav cihazı, **poşetlenmiş steril aletler**, sterilizasyon tezgahı |
| **Detay** | Paketli aletler net görünsün — hijyen hissini bu verir |
| **Kare** | Bir yakın (poşetli aletler) + bir orta plan (tezgah/cihaz) |

## 4. Bekleme Alanı + Dış Cephe

| | |
|---|---|
| **Bekleme** | Koltuklar, ferah bir yatay kare; toplu ve sıcak |
| **Dış cephe** | **Tabela okunacak** şekilde karşıdan, gündüz |
| **Giriş** | Kapı/giriş — hasta mekânı tanıyıp rahat gelsin |
| **Bonus** | Varsa otopark veya "şu binanın yanı" gibi ulaşım ipucu karesi |

## Teslim

Hazır olunca fotoğrafları **e-posta veya WhatsApp (belge olarak)** yolla. Ben:
1. Kırpma, düzleştirme, boyut optimizasyonu (web için hafifletme) yaparım,
2. `assets/photos/` altına yerleştirir, galeri + hekim + ilk-ziyaret bölümlerini kurarım.

> Not: Şu an bu fotoğraflar olmadan bölümleri canlı siteye koymuyorum ki kırık görsel çıkmasın.
> Görseller gelince tek seferde temiz biçimde yayınlarım.
